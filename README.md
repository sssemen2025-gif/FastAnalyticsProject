**Высокопроизводительные аналитические алгоритмы на C++, доступные из Python через pybind11**

---

## О проекте

Я написал этот модуль, чтобы продемонстрировать, как можно существенно ускорить типовые операции анализа данных, вынеся вычисления в нативный C++ код.

Проект учебный, создан в рамках портфолио для демонстрации навыков оптимизации производительности и работы с pybind11.

---

## Возможности

- Скользящее среднее и экспоненциальное сглаживание
- Расчёт медианы, Z-оценок и поиск аномалий
- Линейная регрессия с коэффициентами и R²
- Метрики регрессии (MAE, MSE, RMSE, R²) за один проход
- Поиск локальных пиков
- Значительное ускорение по сравнению с чистым Python
